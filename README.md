# Release Please Template

Этот проект создается пошагово с нуля для демонстрации и настройки процессов контроля качества (governance), CI/CD и автоматизации релизов.

## Шаги создания

1. **Инициализация проекта:** Созданы базовые файлы `package.json` и `README.md` с использованием `pnpm`.
2. **Добавление .gitignore:** Добавлен `.gitignore` файл для исключения `node_modules` и логов pnpm из системы контроля версий.
